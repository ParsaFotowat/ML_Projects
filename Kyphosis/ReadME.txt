# Kyphosis Classification App

The Kyphosis Classification App is a machine learning-based web application built using Streamlit. It allows users to classify kyphosis cases using decision tree and random forest classifiers. Users can either use the default dataset or upload their own CSV file for analysis.

## Features

- Load the default Kyphosis dataset or upload a custom dataset.
- Preview the dataset and visualize it using pairplots.
- Train and evaluate machine learning models (Decision Tree and Random Forest).
- Display confusion matrices and classification reports for model evaluation.

## Requirements

To run the app, ensure you have the following installed:

- Python 3.7 or higher
- Required Python libraries:
  - `streamlit`
  - `pandas`
  - `seaborn`
  - `matplotlib`
  - `scikit-learn`
